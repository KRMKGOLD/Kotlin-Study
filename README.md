## 함수

1. useFunc1.kt

   - 함수 선언

> 오버로딩(overloading) : 같은 이름의 함수를 매개변수 부분을 다르게 하여 여러개 정의하는 기법
>
> 함수는 이름과 매개변수 둘 다 보고 판단하므로, 매개변수가 다르면 선언 중복 에러 발생X
